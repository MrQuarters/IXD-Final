# IXD-Final
Bride to Be - Visual novel made using Gamemaker

Dependancies:
- Gamemaker Studios

Roles:

Director - Anthony Cavuoto

Programming - Andrew Zhou

Artist - Aiyanna Tacco

Story - Javan Reid

Generalist - Ben Greenfield

Controls:
- Spacebar: Confirm/Advance through dialogue
- Arrow Keys(Up and Down): Navigate menu/dialogue choices
- S: Save
- Esc: Exit to main menu

Known Issues:
- Slight pause in dialogue after some dialogue choices
- Game can possibly crash if dialogue is advanced very quickly
- When choosing which location to go to at certain points in the game, only the first option will work. Other options 
have planned out paths, but have not been coded into the game
